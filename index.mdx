Kaku (書く, かく — Japanese for "to write") is a deeply customized terminal emulator built for AI-assisted coding workflows. It ships with sensible defaults, a curated shell suite, and a built-in AI assistant that recovers from errors and generates commands from plain English — all without requiring any initial configuration.

## Why Kaku?

Most terminals force a choice between speed and features, or between good defaults and flexibility. Kaku resolves that tension: it starts fast, looks good, and works well without any setup — while giving you full control through WezTerm's Lua API when you want it.

## Get started

<Steps>
  <Step title="Download and install">
    [Download the latest DMG](https://github.com/tw93/Kaku/releases/latest) and drag Kaku to your Applications folder, or install via Homebrew:

    ```bash
    brew install tw93/tap/kakuku
    ```
  </Step>
  <Step title="Open Kaku">
    Kaku is notarized by Apple and opens without security warnings. On first launch, it automatically sets up your shell environment.
  </Step>
  <Step title="Configure the AI assistant (optional)">
    Run `kaku ai` to open the AI settings panel. Enter your API key and enable the Kaku Assistant for error recovery and command generation.
  </Step>
  <Step title="Customize to your taste">
    Open `~/.config/kaku/kaku.lua` with `kaku config` or `Cmd + ,` to override fonts, themes, keybindings, and more.
  </Step>
</Steps>

<Note>
  Kaku is macOS-only. Windows and Linux support may come in the future.
</Note>
